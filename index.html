Want to see what kinds of errors this utility catches?

Dropping a csv file will run basic validation on it. The first row should be the header row. Based on the name of the header, different validations can be run on the column. This validation is designed to catch typos in entered data, but it is not guaranteed to catch everything!

  • Name validation will be run if the header is named FirstName or LastName
  • Title validation will be run if the header is named Title
  • Email validation will be run if the header is named Email
